# **App Name**: Orbital Date Picker

## Core Features:

- Orbital Drag: Allow users to drag the Earth along its orbit around the sun to select a date.
- Date Display: Display the currently selected date in a human-readable format below the orbit.
- Leap Year Calculation: Ensure that the orbital mechanic and date calculations account for leap years, adjusting the orbit and dates accordingly.
- Date Submission: Allow the selected date to be submitted, triggering the appropriate follow-up action.

## Style Guidelines:

- Primary color: Deep space black (#0B0D17) for the background to emulate space.
- Background color: Slightly lighter deep space black (#161B22). Visible hue should still be space black.
- Accent color: Sun Yellow (#FFC75F) for the sun, creating a strong contrast and drawing attention.
- Font: 'Inter', sans-serif, for clear date display.
- Simple, representational images for the Sun and Earth.
- Center the orbital display with the date displayed directly below it.
- Smooth and responsive dragging animation for the Earth around the orbit.
